# Afia Mbele

### Your Health is Our Priority

# Inspiration

The idea behind Afia Mbele was born from the pressing healthcare challenges across Africa. A major issue is the lack of accessible and reliable healthcare information, compounded by poverty, which prevents a large percentage of the population from receiving proper medical care.

We asked a simple question:
How can technology bridge this gap?

Afia Mbele is our answer — a solution designed to improve access to healthcare information and services across Africa.

# What It Does

Afia Mbele is a web application focused on improving healthcare accessibility through:

## Hospital Locator
Helps users find nearby hospitals and healthcare centers.
## Appointment Booking
Enables better doctor-patient coordination by allowing users to book appointments, saving time and reducing stress.
## Healthcare Information & Support
Guides healthcare subscriptions, insurance, and available medical support systems.
# How We Built It

Afia Mbele was developed using modern and scalable technologies:

Backend: Python, Django
Database: Azure PostgreSQL (Production)
Frontend: HTML, CSS
Cloud & DevOps: Azure
